# My Nurse-Centered AI OS

This folder is your local operating chart for using AI with nursing stewardship.

Core rule: Hermes supports. Humans judge. Nurses steward.

Post-setup suggestion: open `08-Integrations/Local-HTML-Life-Dashboard/index.html` first, create a browser bookmarks folder named **Dashboards**, and bookmark the local dashboard there. Use Notion later as an advanced relational option only if you need database views, filters, relations, or shared command-center structure.

Side-gig doorway: open `03-Community-Entrepreneurship/Nurse-AI-Side-Gig-Starter-Kit/README.md` if you want to validate one safe, no-PHI nurse-AI side project before joining deeper NIN/NAIO pathways.
