
# Operator handoff

The owner should retain:

- the untouched source ZIP and checksum;
- approved Activation Card;
- S0/S1/S2/S3/S4 checkpoint receipts;
- paths for app, database, attachments, logs, exports and backups;
- dependency and license record;
- migration and rollback state;
- test evidence and unresolved blockers;
- final readiness report and exact uninstall instructions.

Never place patient information, credential secrets or restricted institutional content in a receipt, log, screenshot, ticket or support message.
