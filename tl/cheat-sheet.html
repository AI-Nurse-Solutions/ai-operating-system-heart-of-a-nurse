Ang time-out—i-verify bago magsimula

Ang time-out ay paghinto sandali, hindi panic. Ugali ito ng nars: i-check ang setup bago magsimula ang trabaho—tamang machine, tamang folder, tamang permissions, tamang boundaries. Kung may isang hindi pa ready, hindi ibig sabihin na pumalpak ka—natukoy mo lang ang susunod na ligtas na ayusin.

Checked na lahat? Cleared ka na para sa procedure. Hindi pa lahat? Ayusin ang kaya mo, at tandaan—kahit wala ka pang qualifying computer, bukas sa iyo ngayon ang libreng course at 55 AI mentors, ang SOUL Quiz, buwanang Lamp Huddle, at community.

Ang procedure—install at configure

Nagbabago ang menus habang nag-a-update ang Hermes—kapag may mukhang iba, ang official docs ang source of truth.

I-install ang Hermes

Pumunta sa official site—hermes-agent.nousresearch.com—i-download ang para sa computer mo, at sundin ang steps nila. Buksan ito. Kapag tinanong ka ng setup tungkol sa AI model o provider, basahin muna ang susunod na hakbang—may libre at low-cost na routes.

Permissions—bigyan ito ng isang kuwarto, hindi ng buong bahay

Kapag humingi ang Hermes ng file access, access sa My-Nurse-AI-OS folder mo LANG ang ibigay. Tanggihan ang mas malawak na access—full disk, Desktop, Downloads, Photos. Isipin ang badge ng float nurse: para sa assigned unit, hindi sa bawat floor ng ospital. Napasobra ang permission nang hindi sinasadya? Alisin ito sa privacy settings ng computer mo.

Piliin kung paano papaganahin ang AI mo—may libre at low-cost na routes

Libre at open-source ang Hermes mismo. Ang posibleng bayad ay para sa AI model na nagpapatakbo rito—isipin ang Hermes bilang IV pump, libre, at ang model bilang dumadaloy rito, na sinusukat ayon sa paggamit. May dalawang route:

Route A—Nous subscription. Pinakasimple: isang subscription, sila na ang bahala sa lahat. Piliin ito sa setup at laktawan ang natitirang bahagi ng step na ito.

Route B—Bring Your Own Keys. Pinaka-cost-effective, at paraan para patakbuhin ang Hermes nang halos libre. Sa Hermes setup, piliin ang Full Setup (hindi ang default subscription option), pagkatapos ay Bring Your Own Keys. Inirerekomenda namin ang OpenRouter—isang libreng account, isang API key, at maraming model na mapagpipilian, kabilang ang libre at napakamura. Pagkatapos:

  1. Gumawa ng account sa openrouter.ai at mag-generate ng API key mula sa dashboard mo.
  2. Mag-set muna ng spending limit sa OpenRouter—safety rail mo laban sa surprise bills. Magsimula sa ilang dolyar; puwede mo itong taasan anumang oras.
  3. Sa Hermes setup, piliin ang OpenRouter bilang provider at i-paste ang key mo.
  4. Pumili ng model—magsimula sa libre o low-cost. Magtanong ng simple sa Hermes para makumpirmang gumagana ito; kung mahina ang sagot, lumipat sa mas malakas na model. Walang bayad ang pagsubok ng models maliban sa tokens na ginagamit mo.

Isang paalala: karaniwang hindi kasama sa ChatGPT Plus o Grok subscription ang API access—hiwalay ang billing nito. Pinakaligtas ang dedicated OpenRouter key na may spending limit na ikaw mismo ang nagtakda.

Unang conversation mo

Parang chat ang Hermes: mag-type, saka pindutin ang enter. Wala kang masisira sa pakikipag-chat.

Isa akong nars na nagse-set up ng Nurse AI OS ko. Susunod kong ibibigay ang SOUL file ko—gamitin ito bilang standing instructions ko.

Ibigay sa Hermes ang SOUL file mo

Mula sa SOUL Quiz (mada-download bilang Core-SOUL.md, o bilang buong bundle). I-attach, i-drag, o i-paste ito, saka sabihin:

Ito ang SOUL file ko. Gamitin ito bilang standing instructions kung paano mo ako susuportahan. Kumpirmahin kung ano ang naunawaan mo tungkol sa values at boundaries ko.

Pakinggan o basahin ang summary nito. Itama ang anumang mali—mananatili ang corrections.

Ibigay sa Hermes ang Starter Kit mo

I-unzip ang Starter Kit sa lugar na madali mong mahahanap ulit (maganda ang Documents), saka sabihin sa Hermes:

Ang Nurse AI OS folder ko ay nasa [location]. Dito nakatira ang files ko: SOUL file ko, notes ko (Memory), projects ko (Projects), at reusable prompts ko (Skills). Kung may kulang sa mga folder na iyon, gawin ang mga ito. Panatilihin sa loob nito ang lahat ng trabahong ginagawa mo para sa akin.

Buuin ang dashboard mo—at i-bookmark ito

Gamit ang SOUL file ko, i-personalize ang Local HTML Life Dashboard sa starter kit ko (08-Integrations/Local-HTML-Life-Dashboard). Ilagay ang pangalan ko, top priorities ko sa bawat sphere, at mga unang goal at habit mula sa quiz answers ko. I-save ito bilang My-Dashboard.html sa Nurse AI OS folder ko at sabihin sa akin ang eksaktong location nito.

Buksan sa browser ang file na ginawa nito at i-bookmark—iyan ang station board mo.

Bakit mas mahalaga ang SOUL file at Starter Kit kaysa sa lahat: ang makapangyarihang AI ay parang makapangyarihang kabayo—kahanga-hanga, pero delikado kung walang renda. Ang dalawang file na ito ang renda. Ibinibigay ng SOUL file sa engine ang values mo, boundaries mo, at mga linyang hindi nito maaaring tawirin; binibigyan naman ito ng Starter Kit ng tahanan at structure. Magkasama, binibigyan nila ang system ng puso ng isang nars—at pinipigilan ang kabayo na tumakbo nang hindi mo kontrolado. Ang paglaktaw sa mga ito ay hindi nagpapabilis sa iyo; ginagawa ka lang nitong pasahero.
Mas palalimin gamit ang dalawang setup guide: ang Hermes Essentials Guide (students at budget-smart staff—patakbuhin nang halos libre) at The Hermes Power Guide (power users at leaders—profiles, scheduled jobs, model routing). Parehong ipinapakita ang EDENA, ang conscience layer, na gumagana sa architecture mo.

Post-procedure care

Gamitin ang system—hanggang maging muscle memory

Hindi mo kailangang kabisaduhin ang features. Minsan mo lang makikita sa isip ang routine, at susunod na ang mga kamay mo—gaya ng bawat skill na naging automatic na sa iyo. Pitong habit:

1 · I-triage ang task, saka piliin ang model

Simple work—mabilis na tanong, summaries, rewrites, brainstorming—ay para sa libre o murang model. Heavy work—multi-step planning, mahahabang document, coding, malalim na troubleshooting—ay para sa mas malakas na reasoning model. Kailangang tumingin sa screenshot o diagram? Pumili ng vision-capable. Hindi mo tatawagin ang code team para sa tinik: magsimula sa mura, mag-escalate kapag kailangan ng task.

2 · Isang session bawat topic—isang chart bawat pasyente

Huwag pagsamahin sa isang conversation ang budget, side gig, at CCRN plan—naghahalo ang details at lumalabo ang sagot. Mas malinis ang output ng focused session. Bagong topic, bagong session.

3 · Ang Projects ay care plans; ang sessions ay shifts

Ang session ay isang conversation. Para sa anumang tuloy-tuloy—course na binubuo mo, business plan, family project—gumamit ng Project: workspace kung saan magkakasama ang files, goals, at conversations para hindi ka magsimula sa wala.

4 · Tumawag ng consult—gumamit ng personas

Sabihin sa Hermes kung sino ang dapat nitong gampanan, at magbabago ang lalim at style ng sagot: “Kumilos bilang plain-language tutor at ipaliwanag ang error na ito step by step.” · “Kumilos bilang startup strategist at i-review ang plan ko para sa risks at next moves.”

5 · Kapag hindi sigurado, tanungin ang assistant mismo

Hindi mo kailangan ng perfect prompt. “Ano ang pinakamagandang next step?” · “Itanong mo muna ang kailangan mong malaman bago tayo magpatuloy.” · “Hatiin ito sa simpleng steps.” · “Aling model ang dapat kong gamitin para sa task na ito?”

6 · Mag-troubleshoot gamit ang screenshots—ipakita, huwag lang ilarawan

Iabot ang telemetry strip sa halip na ilarawan ito. I-share ang screenshot kasama ang: sinusubukan mong gawin, ano ang nangyari sa halip, eksaktong error, at mga nasubukan mo na. Huwag kailanman isama ang API keys o passwords sa screenshot—i-crop o i-blur muna.

7 · Pangasiwaan ang gastos

Bantayan ang usage gaya ng pagbabantay mo sa supplies: naka-on ang spending limits, drafts sa murang models, strong models para sa final reasoning, at walang napakalaking files maliban kung talagang kailangan ng task.

Ang pang-araw-araw na routine—ulitin hanggang kabisado ng mga kamay mo: tamang model para sa task · bagong session para sa bagong topic · project para sa tuloy-tuloy na work · persona para sa specialist output · magtanong kapag hindi sigurado · magpakita ng screenshot kapag na-stuck · i-escalate lang ang model kapag kailangan.

Habang naghihintay—wala pang qualifying computer? Kasama ka pa rin.

Kung hindi kayang patakbuhin ng kasalukuyang computer mo ang Hermes, installation step lang ang hindi mo pa magagawa. Gumagana sa anumang browser ang libreng AI course at 55 libreng AI mentors, bukas ang buwanang Lamp Huddle at community, at ihahanda ng SOUL Quiz ang SOUL file mo para sa araw na handa na ang system mo.

At malinaw naming sinasabi ito: mahalaga sa amin ngayon ang pakikilahok at boses mo—hindi lang pagkatapos mong bumili ng hardware. Darating ang AI sa healthcare, kasama man o wala ang nursing sa mesa. Ang mga nars na naghahanda ngayon ang mangunguna sa pagbabagong iyon sa halip na sila ang pangunahan nito. Kayo ang mga Florence Nightingale ng hinaharap—kayo ang magpapasya kung paano isisilang ang care intelligence.

Kung naliligaw ka: normal lang. Ang kailangan mo lang ay ang susunod na hakbang, hindi ang buong mapa. Laktawan muna ang nakakalito, at dalhin ang mga tanong sa buwanang Lamp Huddle—nagsimula ang bawat nars doon kung nasaan ka ngayon.
