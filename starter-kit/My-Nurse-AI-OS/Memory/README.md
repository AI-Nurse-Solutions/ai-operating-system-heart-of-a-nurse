# Memory — your working notes live here

This is the folder your AI saves notes, session handoffs, and reviews into. The numbered folders are the library — guides and templates to read. This one is yours to fill.

What belongs here:

- `KARDEX.md` — your digital Kardex: a five-line handoff entry at the end of every meaningful session (what was decided, what was produced, what's still open). Paste it back in when a new session starts.
- `weekly-reviews/` — one short file per week: what worked, what felt unsafe, what you want next week.
- `decisions/` — decisions worth remembering, one per file.

What never belongs here: patient information, employer confidential files, passwords or API keys, or anything that shouldn't resurface later.
