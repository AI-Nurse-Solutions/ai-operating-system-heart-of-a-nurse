Post-setup step

  1. Save this file somewhere simple, such as your Nurse AI OS folder.
  2. Open it in your browser. It works without a server or Notion account.
  3. Create a browser bookmarks folder named Dashboards.
  4. Bookmark this local dashboard under Dashboards so it becomes one click away.
  5. Use broad strokes now. Go deeper later once the dashboard has earned trust.
  6. Export a JSON backup before changing devices or clearing browser data.

Safety boundary

No PHI. No patient identifiers. No employer secrets. No account numbers. No credentials. No diagnosis, treatment, medication, clinical decision support, financial advice, or automated life decisions.

This page is a personal reflection and operations template. If a trend could change health, money, work, family, or clinical decisions, pause and add a human gate.

Human gate

Before sharing, syncing, automating, or acting on any trend, answer:

  1. Could this affect health, money, work, family, or clinical judgment?
  2. Could it expose PHI, employer secrets, account numbers, or credentials?
  3. Do I need a licensed professional, supervisor, mentor, or trusted human to review this?

Agents propose. Humans judge. Nurses steward.
