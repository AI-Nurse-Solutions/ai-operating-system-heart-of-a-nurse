
# Operator handoff

Retain the untouched ZIP and checksum; approved Activation Card; S0–S4 receipts; exact app, database, attachment, log, export, and backup paths; dependency/license record; migration/rollback state; process/port evidence; test evidence; unresolved blockers; final readiness report; and uninstall instructions.

Receipts, screenshots, logs, tickets, and support messages must never contain patient data, real cases, formal evaluations, peer-review/event content, private whole-life content, credential secrets, or restricted institutional material.
