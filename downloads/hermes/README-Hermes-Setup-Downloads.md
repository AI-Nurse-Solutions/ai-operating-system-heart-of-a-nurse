# Hermes + Nurse AI OS Setup Downloads

This folder contains the public download set for nurses who want to set up Hermes Desktop with the Nurse AI OS assets.

## Start here

1. Choose your Hermes Desktop installer:
   - `Hermes-Setup.dmg` for macOS
   - `Hermes-Setup.exe` for Windows
2. Download or unzip `Nurse-AI-OS-Hermes-Setup-Bundle.zip`.
3. Take the SOUL Quiz at https://nurse-ai-os.org/soul-quiz.html.
4. Open the Nurse AI OS starter kit and begin Day 1 of the 30-Day Roadmap.
5. If using the advanced governed build, read https://nurse-ai-os.org/naio-os/README.md first.

## Safety boundary

No PHI. Do not paste patient names, charts, screenshots, identifiers, room numbers, employer secrets, or patient-specific clinical details into Hermes, the quizzes, or any AI workspace.

## Important distinction

Hermes Desktop is the runtime. Nurse AI OS is the governed no-PHI learning and stewardship layer we provide around it. The macOS and Windows installers make Hermes easier to reach; the governed NAIO harness remains human-reviewed, dry-run-first, and bounded by the Nurse AI OS setup guides.

Agents propose. Humans judge. Nurses steward.
