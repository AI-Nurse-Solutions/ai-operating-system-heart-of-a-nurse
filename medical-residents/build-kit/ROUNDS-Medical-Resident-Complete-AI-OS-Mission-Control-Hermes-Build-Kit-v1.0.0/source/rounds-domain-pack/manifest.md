## Compatibility and non-dependency

This program does not depend on, modify, or share state with any nursing population OS. A separate Medical Resident workspace is required when isolation cannot be proven. The only supported import is an explicit, field-level, low-risk personal preview into the isolated resident lane. Clinical, evaluation, authority, credential, institutional, patient, and agent data do not migrate.
