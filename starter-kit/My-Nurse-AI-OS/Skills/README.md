# Skills — reusable prompts your system learns

A skill is a prompt you'll use more than once, saved as a small markdown file so your AI can load it instead of you retyping it. When a prompt works, save it here and it becomes part of your system.

Suggested layout:

```text
Skills/
├── personal/        e.g. daily-reflection.md
├── professional/    e.g. socratic-tutor.md, meeting-prep.md
└── community/       e.g. voice.md
```

Turn any good moment into a skill with: *"That worked well. Save it as a reusable skill in Skills/[folder]/[name].md — include when to use it and the exact prompt."*

Review your skills every 90 days; archive what you no longer use. No patient data, no employer secrets, no credentials inside skill files.
