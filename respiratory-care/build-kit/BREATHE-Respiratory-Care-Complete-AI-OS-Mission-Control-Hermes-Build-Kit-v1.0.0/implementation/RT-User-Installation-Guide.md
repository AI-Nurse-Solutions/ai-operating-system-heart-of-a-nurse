
# User Installation Guide

1. Keep the original ZIP unchanged.
2. Verify its SHA-256 sidecar.
3. Extract a private working copy.
4. Run the embedded verifier.
5. Give the whole directory to Hermes and use `README-FIRST.md`.
6. Wait for the read-only preflight and inspect the Activation Card.
7. Approve, revise or cancel the exact card. Approval does not authorize patient data, clinical work, connectors, messages, external actions or agents.
8. Keep S0/S1/S2/S3/S4 receipts. The installation may take multiple turns.
9. Use only fictional, public or screened low-sensitivity content for first-run tests.
10. Accept the application only after the final evidence-based readiness report.

After installation, begin with the Core Four. Preview no more than one BREATHE power in the first week. Use Pause All whenever state, data, authority, device boundary or destination is uncertain.
