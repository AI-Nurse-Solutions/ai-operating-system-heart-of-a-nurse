
# Personalization inputs

The three bundled JSON files are synthetic demonstrations of strict adapters. They contain no personal facts and are not badge evidence. A real first run must derive minimum fields in memory, show provenance and uncertainty, and persist only fields the owner approves.

Do not copy raw Discover Packet answers, Soul Quiz answers or raw `SOUL.md` into the application. Do not infer licensure, competence, authority, diagnosis, credential status or employment facts.
