# Projects — one folder per ongoing effort

A session is one conversation; a project is a care plan. Anything ongoing — a course you're building, a certification plan, a side gig, a family project — gets its own folder here so files, goals, and notes stay together and you never start from scratch.

Pattern:

```text
Projects/
├── ccrn-plan/
├── unit-education-series/
└── side-gig-validation/
```

Start each project by asking your AI: *"Create a new project folder in Projects called [name]. Interview me like an admission assessment, then write a one-page project brief inside it."*

No patient data, ever — projects run on your life and work, never your patients.
