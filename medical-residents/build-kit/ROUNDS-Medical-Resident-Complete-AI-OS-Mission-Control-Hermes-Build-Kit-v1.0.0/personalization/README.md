
# Personalization inputs

The bundled JSON files are synthetic demonstrations of strict adapters. They contain no personal facts, authority, or badge evidence. A real first run derives minimum fields in memory, displays every field with provenance and uncertainty, and persists only owner-approved fields.

Never copy raw Discover answers, Soul Quiz answers, raw `SOUL.md`, patient/event content, formal evaluations, or authority claims into the application. Program, PGY, role, attending, and supervision values remain unverified until an authorized external source and human process establish otherwise.
