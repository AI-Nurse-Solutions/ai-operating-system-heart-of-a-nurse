# process-learning-inbox Skill Draft

Use only after the workflow has proven useful 2–3 times.

Purpose: Convert raw learning inputs into structured, evidence-aware, nurse-reframed markdown notes using the HERMES Transformation Protocol.

Trigger: User provides an article, transcript, podcast notes, lecture notes, paper, or link.

Output: Markdown learning note with source understanding, evaluation, nursing reframe, systems map, enabled action, and stewardship review.

Boundaries:

- No PHI
- No patient-specific recommendations
- No clinical advice
- Name uncertainty and what requires human verification
