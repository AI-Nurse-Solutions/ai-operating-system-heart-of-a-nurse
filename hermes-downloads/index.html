Hermes Install Guide

Hermes is the free, open-source AI engine your Nurse AI OS runs on. This page is only about getting it installed — from the official site, safely, on your computer. Nothing here replaces your onboarding.

Your whole setup is the five steps on Start Here — this page is the optional deep dive for Step 1, installing Hermes from the official site. If installation went smoothly for you, you don't need this page at all. When Hermes is running, return to Start Here and continue with Step 2.

The one rule of installation

Always install from the official Hermes site.

Nurse AI OS does not host Hermes installers, and neither should anyone else. Mirrored copies go stale, break, and can't be trusted the way the source can. Get Hermes directly from Nous Research — the team that builds and maintains it — and you'll always have the current version and the current instructions.

Get Hermes (official site) →  Official installation docs

Hermes is open source — you can read the code on GitHub anytime. Prefer paper at your elbow? The one-page cheat sheet covers install and first launch.

Platform by platform

Install details for your computer

Menus and buttons move as Hermes updates — when this page and the official docs disagree, the official docs are the truth. These notes cover what stays constant.

Mac

Download the Mac installer from the official site and open it like any other app. If macOS asks whether you're sure about an app downloaded from the internet, that's a standard prompt — confirm it only because you downloaded it from the official site yourself.

First launch: if Hermes asks about an AI model or provider, the default is fine — you can change it later.

Windows

Download the Windows installer from the official site and run it. If Windows shows a security prompt for a newly downloaded app, the same rule applies: proceed only because the download came straight from the official site.

First launch: accept the defaults, open Hermes, and say hello. It works like a chat — you cannot break anything.

Your first conversation

Type like you'd text a colleague. A good opener: "I'm a nurse setting up my Nurse AI OS. I'll give you my SOUL file next — treat it as my standing instructions." (Your SOUL file is Step 2, back on Start Here.)

The phrase that keeps your hand on every decision: "Show me your plan before you do anything."

If something goes wrong

Check the official installation docs first — they track the current version. Still stuck? Bring it to the monthly Lamp Huddle or email great.ai.nurses@gmail.com — a human answers.

Rather not install anything yet? You can do steps 2–5 in the browser AI you already use (ChatGPT or Claude) — paste instead of upload. Hermes just makes it permanent.

Plain language

What Hermes is — and what Nurse AI OS adds around it

Hermes Agent is an open-source AI agent framework by Nous Research. Depending on how you configure it, it can use memory, skills, tools, files, and automations. Nurse AI OS is not a replacement for Hermes — it's the nurse-led layer you build around it.

Engine

Hermes is the runtime

Think of Hermes as the engine that can read, write, search, remember, and carry out tasks when a human permits it.

Stewardship

Nurse AI OS is the governed layer

Nurse AI OS gives Hermes a nursing-centered purpose: no-PHI boundaries, your SOUL file, EDENA gates, dashboards, prompts, and starter folders.

Human gate

The nurse remains the judge

Hermes may propose, organize, summarize, or draft. The nurse decides what is true, safe, appropriate, and ready to use.

The simplest metaphor: Hermes is the capable open-source assistant. Nurse AI OS is the nurse-led operating discipline that tells the assistant what it may do, what it must never do, and how to serve human dignity.

The safety perimeter

No PHI. No clinical decisions. No silent automation.

Do not enter patient names, charts, screenshots, identifiers, room numbers, employer secrets, passwords, account numbers, or patient-specific clinical details into Hermes, the quizzes, dashboards, or any AI tool — unless an approved institutional environment and accountable governance explicitly allow it.

Hermes can help with learning, organization, drafting, research support, project planning, and reflection. It does not carry a license, validate competency, approve clinical deployment, replace educators, or make patient-care decisions.

Agents propose. Humans judge. Nurses steward. Gray areas? Bring them to the monthly Lamp Huddle — a question asked early beats an incident reported late.
