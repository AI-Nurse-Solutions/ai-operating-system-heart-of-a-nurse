# Google Workspace HIPAA and Safe AI Use

This folder explains why Google Workspace is a governed suite, not just email, and how nurses/faculty should think about PHI, BAAs, personal Gmail, Workspace core services, and external AI tools.

Default course posture:

> Hermes is no-PHI unless deployed under explicit institutional governance.

Core message:

> BAA-backed tools can create a safer room. Nurses still need rules for what enters the room, who can see it, and what AI is allowed to do with it.
