# STEWARD Governance Preview

## Hospital & Clinic Administration

**Non-executable · Not institution-approved · No operational data**

This directory contains the public STEWARD governance preview. It preserves the administrative doctrine and one-lane/context-variant architecture while the installable **Complete AI OS** claim remains paused.

- `STEWARD-Governance-Specification.md` — canonical public doctrine and architecture
- `STEWARD-Governance-Specification.pdf` — publication-formatted version
