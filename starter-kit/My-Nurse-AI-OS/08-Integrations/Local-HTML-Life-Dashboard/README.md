# Local HTML Life Dashboard

Use this fallback when a Notion dashboard is not possible, too heavy, blocked by workplace policy, or simply not the right tool yet.

Open `index.html` in a browser to use a **browser-only, no-PHI life dashboard** for:

- Health & Wellbeing
- Finances
- Goals
- Habits
- Tasks
- Routines

The template includes Today, This Week, Health & Energy, Money Stewardship, Goals & Projects, and Weekly Reset views. It stores edits in your browser's local storage and can export a private JSON backup.

## Post-setup suggested step

1. Save or keep this folder inside your Nurse AI OS starter kit.
2. Open `index.html` in your browser.
3. Create a browser bookmarks folder named **Dashboards**.
4. Bookmark the local dashboard under **Dashboards** so it becomes one click away.
5. Use broad strokes now; go deeper later once the dashboard has earned trust.
6. Export a JSON backup before clearing browser data or changing devices.

## Boundary

This is a personal operations cockpit. It is not a clinical system, EHR, diagnosis tool, treatment tracker, financial adviser, therapy tool, performance evaluation system, or automated decision-maker.

Do not enter PHI, patient identifiers, employer secrets, account numbers, credentials, diagnosis, treatment, medication, clinical decision support, or confidential work/family details.

## When to choose this over Notion

Choose the local HTML dashboard when:

- you cannot access Notion,
- your organization blocks Notion,
- you want the simplest no-account path,
- you need a private local dashboard before building a relational database,
- or the best first step is just to begin gently.

Agents propose. Humans judge. Nurses steward.
